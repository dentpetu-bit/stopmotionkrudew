# Stop Motion Studio Classroom By Kru Dew - V8

เวอร์ชันนี้แก้ปัญหา Export GIF ขึ้น `drawImageContain is not defined` และคงสัดส่วนภาพ GIF ให้ไม่ยืด/ไม่บิด


เว็บแอปสำหรับให้นักเรียนวาดภาพทีละเฟรมบน iPad, Tablet, มือถือ หรือคอมพิวเตอร์ แล้วรวมเป็น Stop Motion ได้

## ไฟล์ในโปรเจกต์

- `index.html` หน้าเว็บหลัก
- `style.css` ไฟล์ตกแต่งหน้าตาเว็บ
- `app.js` ระบบวาดภาพ บันทึกเฟรม Timeline Preview Export GIF และ Admin
- `supabase.js` ตั้งค่า Supabase URL, anon key และรหัสผ่านครู
- `supabase.sql` SQL สำหรับสร้างฐานข้อมูลและ Storage policy
- `README.md` คู่มือใช้งาน

## วิธีติดตั้ง Supabase

### 1. สร้าง Project

เข้า Supabase แล้วสร้าง Project ใหม่

### 2. รัน SQL

ไปที่เมนู SQL Editor แล้ววางโค้ดจากไฟล์ `supabase.sql` จากนั้นกด Run

ระบบจะสร้างตาราง:

- `students`
- `projects`
- `frames`

และสร้าง Storage bucket ชื่อ:

- `stop-motion-frames`

### 3. ตรวจสอบ Storage Bucket

ไปที่เมนู Storage แล้วตรวจสอบว่ามี bucket ชื่อ `stop-motion-frames`

ตั้งค่าให้เป็น Public bucket เพื่อให้ภาพเฟรมแสดงบนเว็บได้

### 4. ตั้งค่า Supabase ในโค้ด

เปิดไฟล์ `supabase.js` แล้วแก้ไขค่าต่อไปนี้

```js
const SUPABASE_URL = "YOUR_SUPABASE_URL";
const SUPABASE_ANON_KEY = "YOUR_SUPABASE_ANON_KEY";
```

หาได้จาก:

Supabase Dashboard > Project Settings > API

แล้วคัดลอก:

- Project URL
- anon public key

### 5. เปลี่ยนรหัสผ่าน Admin

ในไฟล์ `supabase.js`

```js
const ADMIN_PASSWORD = "kru-dew-1234";
```

เปลี่ยนเป็นรหัสที่ครูต้องการ

หมายเหตุ: ระบบนี้เป็น GitHub Pages จึงเป็นรหัสผ่านแบบง่าย ไม่เหมาะกับข้อมูลลับมาก ๆ

## วิธีอัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลดไฟล์ทั้งหมดเข้า Repository
3. ไปที่ Settings > Pages
4. เลือก Branch เป็น `main`
5. เลือก Folder เป็น `/root`
6. กด Save
7. รอ GitHub สร้างลิงก์เว็บให้

## วิธีใช้งานสำหรับนักเรียน

1. เปิดเว็บ
2. กด “เริ่มสร้าง Stop Motion”
3. กรอกรหัสนักเรียน ชื่อ ห้อง ชื่อโปรเจกต์
4. กดสร้างโปรเจกต์
5. วาดภาพเฟรมแรก
6. กด “บันทึกเฟรม”
7. วาดภาพเฟรมถัดไป โดยใช้ Onion Skin ช่วยดูภาพก่อนหน้าแบบจาง ๆ
8. กด Preview เพื่อดูภาพเคลื่อนไหว
9. ปรับ FPS ได้ เช่น 2, 5, 10, 12 fps
10. กด Export GIF เพื่อดาวน์โหลดผลงาน

## วิธีใช้งานสำหรับครู

1. กดปุ่ม Admin
2. ใส่รหัสผ่านครู
3. ดูจำนวนโปรเจกต์ จำนวนเฟรม และจำนวนนักเรียน
4. ค้นหาผลงานตามชื่อ ห้อง หรือชื่อโปรเจกต์
5. เปิดดูผลงานนักเรียน
6. ดาวน์โหลดข้อมูลเป็น CSV
7. ลบผลงานที่ไม่เหมาะสมได้

## ข้อจำกัด

- Export หลักเป็น GIF เพราะทำงานบน Browser และ GitHub Pages ได้ง่ายกว่า MP4
- ถ้าเฟรมมีจำนวนมาก GIF อาจใช้เวลาสร้างนาน
- ถ้าอินเทอร์เน็ตช้า การอัปโหลดเฟรมอาจช้า
- รหัสผ่าน Admin อยู่ในฝั่งหน้าเว็บ จึงไม่ใช่ระบบรักษาความปลอดภัยขั้นสูง

## แก้ปัญหาเบื้องต้น

### ภาพไม่ขึ้น

ตรวจสอบว่า bucket `stop-motion-frames` เป็น Public แล้วหรือไม่

### บันทึกเฟรมไม่ได้

ตรวจสอบว่าใส่ Supabase URL และ anon key ถูกต้องหรือไม่

ตรวจสอบว่าได้รัน `supabase.sql` แล้วหรือไม่

### วาดบน iPad แล้วหน้าเว็บเลื่อน

ระบบตั้งค่า `touch-action: none` แล้ว แต่ควรใช้ Safari หรือ Chrome รุ่นใหม่

### Export GIF ไม่ได้

ตรวจสอบอินเทอร์เน็ต เพราะระบบใช้ไลบรารี GIF จาก CDN
